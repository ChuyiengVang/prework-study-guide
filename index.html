HTML

The structure of a webpage

Elements

!DOCTYPE html - defines what page is
head - container for metadata
meta - inofrmation about the page
charset - sets character encoding
http-equiv - sets an http header for content
viewport - sets default view for webpage
body - what you can see on the page
div - create division in the page
span - inline division
ul - unodered list
ol - ordered lists
li - list items
img - info about images
br - line break(no need for closing tag)
a - creates link to webpages
h1-h5 - headings large to small
main - represents main content of body
footer - info that appears at bottom of page
section - semantic elements that define a section
link - connects to external file

Attributes

src - define location of the image
alt - alternate info of image
href - points to the URL
id - a way to identify elements
rel - specifies relationship current
doc and linked doc resource

CSS

Dictates HTML

Property

margin - indicates how much space around the
outside of an element
padding - adds space inside an element
border - assigns a border
width - width value
box-shadow - adds a shadow effect

Git

  • git add -A - commit all files in the working branch
  • git checkout -b branch-name - creates a new branch and switches to it
  • git checkout main - switch to main branch
  • git status - check current location
  • git commit -m "message" - descriptive message of the changes implemented
  • git pull origin main - pulls in latest changed main branch
  • git push origin file- push changes

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • String - included topics
  • Array - group of related data
  • Index - unique numbered location of data item in the array
  • For Loops - allows a single code to be executed repeatedly
